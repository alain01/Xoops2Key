Adminer

CatégorieAdministration
Version3.1 Beta 1
XOOPS 2.5.X
PHP 5.3+
PHP 7

Le module Adminer est un module qui propose des outils de gestion de la base de données SQL directement dans le panneau de configuration. Il inclue 3 outils :
- Adminer
- Adminner editor
- Bigdump, script permettant d'importer de grosses bases de données

wgXPiwik

CatégorieAdministration
Version1.02
XOOPS 2.5.X
PHP 5.3+
PHP 7

Le module wgXPiwik est une implémentation de Piwik dans XOOPS. Statistiques et analyses des visites.
Gestion :
- des permissions par groupes
- Suivi dans des blocks, par module ou pour tout les modules

Publisher

CatégorieContenu
Version1.03 Final
XOOPS 2.5.X
PHP 5.3+
PHP 7

Le module Publisher est un module qui permet de gérer des articles et des blogs. Facilement clonable. SEO.
Gestion :
- des persmissions
- des catégories et sous-catégories
- date de publication et d'expiration
